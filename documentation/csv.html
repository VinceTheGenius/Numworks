Initialisation

Déjà ce qui est important à savoir c'est que l'objet CSV ne peut gérer qu'une entrée sous forme de texte. Avec une virgule pour séparer les éléments et un retour à la ligne pour séparer les lignes.

Il y a trois moyens d'ouvrir d'initialiser un objet CSV.

Tout d'abords on peut mettre le texte directement dans le script.

variable="""
Nom,numéro,date_de_naissance
Claude,0618493827,2004
Gérard,0649382793,2006
Léa,0792649355,2003
...
"""
instance=Csv(variable)

Deuxièmement on peut mettre le texte dans un fichier et importer le fichier dans le script charger de l'interpréter.

variable="""
Nom,numéro,date_de_naissance
Claude,0618493827,2004
Gérard,0649382793,2006
Léa,0792649355,2003
...
"""

import csv_txt
instance=Csv(csv_txt.variable)

Et dernièrement on peut ouvrir un fichier python sans que le texte soit encodé dedans. Mais ATTENTION, cela ne fonctionne que sur Omega.

Nom,numéro,date_de_naissance
Claude,0618493827,2004
Gérard,0649382793,2006
Léa,0792649355,2003
...

instance=Csv()
instance.open("csv_txt.py")
